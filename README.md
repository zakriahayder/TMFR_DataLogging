# Vehicle Telemetry Console

Real-time vehicle telemetry monitoring dashboard built with Next.js, React, and TypeScript.

## Setup

```bash
pnpm install
```

## Commands

- `pnpm dev` - Start development server at `http://localhost:3000`
- `pnpm build` - Build for production
- `pnpm start` - Run production server
- `pnpm lint` - Run ESLint

## Tech Stack

Next.js 16, React, TypeScript, Radix UI, Tailwind CSS, Recharts
